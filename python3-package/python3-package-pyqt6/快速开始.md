# PyQt6 新增案例 - 快速开始指南

## 📦 依赖安装

### 方法一：一键安装所有依赖（推荐）

```bash
pip install -r requirements.txt
```

### 方法二：按需安装

#### 核心依赖（必需）
```bash
pip install PyQt6 PyQt6-sip
```

#### 可选依赖（根据使用的案例安装）

**网络请求工具** 需要：
```bash
pip install requests
```

**系统监控面板** 需要：
```bash
pip install psutil
```

**数据导入导出工具** 需要：
```bash
pip install pandas openpyxl
```
