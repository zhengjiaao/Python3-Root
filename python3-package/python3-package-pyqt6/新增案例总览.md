# PyQt6 新增案例总览

本文档介绍了为 PyQt6 新创建的 9 个真实案例，涵盖基础操作、高级功能和实际应用场景。

## 📋 案例清单

### 一、基础操作（3个新案例）

#### 1. 文件管理器 (`基础操作/4 文件管理器.py`)
**功能特点：**
- 双视图设计：左侧树形目录结构 + 右侧文件列表
- 地址栏导航，支持直接输入路径
- 工具栏：上级目录、刷新、新建文件夹、删除
- 支持文件和文件夹的浏览、选择、删除操作
- 状态栏显示当前路径和操作反馈

**学习要点：**
- QTreeView 和 QListView 的使用
- QFileSystemModel 文件系统模型
- QSplitter 分割器布局
- 文件系统的增删改查操作

---

#### 2. 文本编辑器 (`基础操作/5 文本编辑器.py`)
**功能特点：**
- 完整的文本编辑功能：新建、打开、保存、另存为
- 丰富的格式设置：字体、字号、加粗、斜体、下划线
- 颜色设置：文字颜色、背景颜色
- 撤销/重做、剪切/复制/粘贴
- 光标位置实时显示
- 修改检测，关闭时提示保存

**学习要点：**
- QTextEdit 富文本编辑
- QTextCharFormat 字符格式设置
- QFontComboBox 字体选择
- 文件读写操作
- 文档修改状态追踪

---

#### 3. 图片浏览器 (`基础操作/6 图片浏览器.py`)
**功能特点：**
- 支持多种图片格式：PNG、JPG、JPEG、BMP、GIF
- 缩放功能：放大、缩小、适应窗口、原始大小
- 旋转功能：左转90°、右转90°、自定义角度
- 翻转功能：水平翻转、垂直翻转
- 滑块控制缩放比例（10% - 500%）
- 保存图片到指定格式

**学习要点：**
- QGraphicsView 和 QGraphicsScene 图形视图框架
- QPixmap 图片处理
- 坐标变换：缩放、旋转、翻转
- 文件对话框使用

---

### 二、高级功能（3个新案例）

#### 4. 网络请求工具 (`高级功能/3 网络请求工具.py`)
**功能特点：**
- 支持 HTTP 方法：GET、POST、PUT、DELETE
- Headers 管理：可添加、编辑、删除请求头
- Body 编辑：支持 JSON 数据输入
- 响应展示：
  - 原始响应文本
  - JSON 格式化显示
  - 响应 Headers
- 状态码和耗时显示
- 多线程异步请求，不阻塞UI

**学习要点：**
- requests 库的使用
- QThread 多线程编程
- pyqtSignal 信号槽机制
- JSON 数据处理
- 表格控件 QTableWidget

**依赖安装：**
```bash
pip install requests
```

---

#### 5. 定时任务调度器 (`高级功能/4 定时任务调度器.py`)
**功能特点：**
- 任务管理：添加、启动、停止、删除任务
- 灵活的执行间隔：秒、分钟、小时
- 任务状态实时监控：待启动、运行中、已停止
- 执行日志记录：开始时间、完成时间、错误信息
- 最后执行时间显示
- 关闭窗口时自动停止所有任务

**学习要点：**
- QTimer 定时器使用
- 多线程任务执行
- 任务生命周期管理
- 动态表格操作（添加按钮控件）
- 资源清理和线程安全

---

#### 6. 数据导入导出工具 (`高级功能/5 数据导入导出工具.py`)
**功能特点：**
- 支持格式：CSV、Excel (.xlsx)
- 导入功能：
  - CSV 文件导入
  - Excel 文件导入（需要 pandas）
- 导出功能：
  - 导出为 CSV（UTF-8 with BOM）
  - 导出为 Excel（需要 pandas）
- 表格编辑：添加行、删除选中行、清空表格
- 数据统计：行数、列数实时显示
- 操作日志记录

**学习要点：**
- csv 模块使用
- pandas 和 openpyxl 集成
- QTableWidget 数据操作
- 文件编码处理（UTF-8 BOM）
- 异常处理和用户提示

**依赖安装：**
```bash
pip install pandas openpyxl
```

### 三、应用场景（3个新案例）

#### 7. 待办事项管理 (`应用场景/1 待办事项管理.py`)
**功能特点：**
- 完整的 CRUD 操作：创建、查看、更新、删除
- 优先级管理：低、中、高三级
- 截止日期设置
- 完成状态切换
- 数据库持久化存储（SQLite）
- 统计信息：总计、已完成、待完成
- 可选显示/隐藏已完成任务
- 详细描述字段

**学习要点：**
- SQLite 数据库操作
- sqlite3 模块使用
- 数据库类封装设计
- QListWidget 自定义显示
- 数据与UI分离架构

---

#### 8. 系统监控面板 (`应用场景/2 系统监控面板.py`)
**功能特点：**
- CPU 监控：
  - 实时使用率进度条
  - 核心数、频率显示
  - 颜色预警（绿/橙/红）
- 内存监控：
  - 使用率进度条
  - 总计、已用、可用容量
- 磁盘监控：
  - 各分区使用情况表格
  - 容量统计
- 进程监控：
  - Top 10 CPU 占用进程
  - PID、名称、CPU%、内存%、状态
- 网络统计：
  - 发送/接收字节数
- 自动刷新（每2秒）

**学习要点：**
- psutil 系统信息库
- QProgressBar 样式定制
- QTimer 定时更新
- 数据格式化（字节转可读单位）
- 实时数据可视化

**依赖安装：**
```bash
pip install psutil
```

---

#### 9. 批量文件重命名 (`应用场景/3 批量文件重命名.py`)
**功能特点：**
- 6种重命名模式：
  1. **添加前缀**：如 `[重要]文档.pdf`
  2. **添加后缀**：如 `文档_backup.pdf`
  3. **替换文本**：查找并替换文件名中的文本
  4. **删除字符**：前N个、后N个、指定位置
  5. **序号重命名**：自定义前缀+序号（001、0001等）
  6. **正则表达式**：强大的模式匹配替换
- 预览功能：执行前查看重命名结果
- 支持单个文件和文件夹批量添加
- 文件计数和状态显示
- 安全确认：执行前二次确认

**学习要点：**
- os 模块文件操作
- re 正则表达式
- QComboBox 动态界面切换
- 字符串处理和模式匹配
- 批量操作和错误处理

## 📦 依赖安装

所有案例的依赖已在 `requirements.txt` 中配置：

```bash
pip install -r requirements.txt
```

或单独安装：

```bash
# 核心依赖
pip install PyQt6 PyQt6-sip

# 可选依赖（根据需要使用）
pip install requests      # 网络请求工具
pip install psutil        # 系统监控面板
pip install pandas        # 数据导入导出（Excel）
pip install openpyxl      # 数据导入导出（Excel）
```
